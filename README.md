# vuejs-tutorial

## 1) Vue setup

### Install CLI
```
npm install -g @vue/cli
```

### Create project
```
vue create <project_name>
```

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

## 2) Vue-router setup

### Install vue-router(3.x)
```
npm install vue-router
```
- need to add <router-view /> in app.vue for access routes
